# AI HR Assistant

## Описание проекта

Загружаете резюме кандидатов и описание вакансии — AI анализирует соответствие навыков, опыта и требований, выставляет скор совместимости и автоматически ранжирует кандидатов. Для лучших кандидатов система генерирует персональные вопросы для интервью, основанные на их опыте и пробелах в навыках. Интегрируется с популярными HR-системами.

## Цель проекта

Ускорение процесса отбора кандидатов и повышение качества найма через автоматизацию первичной оценки.

## Функциональность

1. Парсинг и анализ резюме (PDF, Word, текст)
1. Скоринг соответствия кандидатов вакансии
1. Автоматическое ранжирование кандидатов
1. Генерация персональных вопросов для интервью
1. Выявление ключевых навыков и пробелов

# Product Requirements Document

## Сегменты пользователей:

Основатель стартапа или директор небольшой компании (5–30 сотрудников), который сам занимается наймом. Проект только стартует, HR-отдела нет, нанимать приходится лично. Руководитель часто не является экспертом в сфере, куда ищет сотрудников. Например:
- маркетолог ищет backend-разработчика;
- бизнес-аналитик нанимает ML-инженера;
- менеджер по продажам подбирает дизайнера.

## Pain Points:

1. Неопытность в собеседованиях
	- Нет опыта в структурировании интервью.
	- Сложно отличить сильного специалиста от посредственного, особенно если оба уверенно говорят.

1. Некомпетентность в области найма
	- Руководитель не знает технических нюансов профессии и боится задавать “неправильные вопросы”.
	- Есть риск нанять человека, который выглядит подходящим на словах, но не умеет решать реальные задачи.

1. Отсутствие HR-экспертизы
	- Нет методик оценки soft skills, культурного соответствия.
	- Нет шаблонов вопросов и процессов для проведения интервью.

1. Ограниченное время
	- Владелец совмещает 5 ролей в стартапе и не может тратить часы на подготовку к каждому собеседованию.

1. Высокая цена ошибки
   
## Анализ текущих решений

1. Крупные ATS-системы\
Представители: Хантфлоу, HR-ONLINE, Проходчик, Talantix
- Плюсы:
	- Интеграция с популярными HR-порталами (hh.ru, Habr Career)
	- Поддержка электронного документооборота (ЭДО)
	- Локализованная отчетность
	- Онбординг и управление испытательным сроком
- Минусы:
	- Стоимость от 1000-5000 руб/месяц за пользователя
	- Сложность настройки под нужды малого бизнеса
	- AI-функционал практически отсутствует или очень слабый
	- Устаревший пользовательский интерфейс
	- Долгое внедрение и обучение

	**Чем ползен наши проект:** мгновенный старт без внедрения и сложных настроек, фокусируясь только на анализе кандидатов и подготовке к интервью.

2. Платформы-агрегаторы вакансий с AI-поиском\
Представители: hh.ru (AI-подбор), Avito Работа, Habr Career
- Плюсы:
	- Огромная база кандидатов
	- Встроенные инструменты для публикации вакансий
	- Простой интерфейс для коммуникации
	- Мобильные приложения
	- Дополнительные сервисы (тестирование, проверка рекомендаций)
- Минусы:
	- Поверхностный анализ кандидатов
	- Нет глубинной аналитики соответствия
	- Ограниченный функционал для работы с резюме
	- Нет персонализированной помощи в проведении интервью
	- Фокус на количестве, а не качестве подбора

   **Чем ползен наши проект:** глубинная аналитика по каждому кандидату и персональные вопросы для интервью, а не просто фильтрация по ключевым словам.

3. Специализированные сервисы для проверки кандидатов\
Представители: Скайпро, Yva.ai, Куншт, HR-Лаб
- Плюсы:
	- Глубокая аналитика компетенций
	- Психологическое тестирование
	- Проверка мягких навыков (soft skills)
	- Анализ культурного соответствия
	- Отслеживание вовлеченности сотрудников
- Минусы:
	- Высокая стоимость для малого бизнеса
	- Узкая специализация (только тестирование)
	- Не покрывают полный цикл подбора
	- Требуют интеграции с другими системами
	- Сложная интерпретация результатов для не-HR

	 **Чем ползен наши проект:** комплексный анализ на этапе отбора резюме с простой интерпретацией результатов для не-специалистов.
  
4. Конструкторы тестов и собеседований\
Представители: Testograf, iSpring, Экзамус, Профилум
- Плюсы:
	- Создание профессиональных тестов
	- Автоматическая проверка результатов
	- Готовые шаблоны для разных профессий
	- Анализ прогресса кандидатов
	- Интеграция с другими системами
- Минусы:
	- Только этап оценки, нет полного цикла
	- Не анализируют резюме и опыт
	- Не помогают с подготовкой к интервью
	- Ограничены форматом тестирования
	- Не дают комплексной картины о кандидате

	**Чем ползен наши проект:** работаем с резюме без создания тестов и готовим вопросы для живого диалога, а не для автоматической проверки.

5. Самодельные решения на базе ChatGPT/Yandex GPT
- Плюсы:
	- Бесплатно или очень дешево
	- Гибкость в настройке запросов
	- Работа с русским языком на высоком уровне
	- Мгновенное получение результатов
	- Не требует установки дополнительного ПО
- Минусы:
	- Ручной перенос данных между системами
	- Нет накопления базы знаний и истории
	- Проблемы с конфиденциальностью персональных данных
	- Нестабильность качества ответов
	- Требует постоянной тонкой настройки промптов

	
	**Чем ползен наши проект:** безопасный и структурированный процесс с защитой данных и предсказуемым качеством результатов.

## Use Cases

### UC1: Создание вакансии и загрузка кандидатов

**Акторы:** пользователь (директор/основатель), система (AI HR Assistant), LLM (внешняя система)\
**Предусловия:** пользователь авторизован; интернет соединение активно, у пользователя есть готовое описание вакансии

Сценарий:
1. Пользователь загружает описание вакансии (документ или текст)
1. Система парсит и сохраняет требования вакансии
1. Пользователь загружает пакет резюме кандидатов
1. Система подтверждает успешную загрузку и начинает обработку

User Value:
- Экономия времени на формализацию требований
- Единая точка управления кандидатами

### UC2: Выбор варианта поиска и настройка критериев
**Акторы:** пользователь (директор/основатель), система (AI HR Assistant)\
**Предусловия:** вакансия успешно создана и сохранена в системе, загружено минимум 1 резюме кандидата, система готова к анализу данных

Сценарий:
1. Система предлагает выбрать вариант анализа:
    - По умолчанию - автоматический расчет с равными весами критериев
    - По критериям - с возможностью настройки приоритетов
1. Пользователь выбирает вариант анализа:
    - Если выбран вариант "По умолчанию":
        - Система применяет стандартные настройки весов
        - Переход к UC3
    - Если выбран вариант "По критериям":
        - Система отображает список критериев из вакансии
        - Пользователь назначает веса критериям
        - Пользователь подтверждает настройки

User Value:
- Контроль над процессом оценки - пользователь может выделить наиболее важные для него критерии
- Возможность выбрать между быстрым анализом "из коробки" и тонкой настройкой под специфичные требования


### UC3: Анализ соответствия и ранжирование кандидатов
**Акторы:** пользователь (директор/основатель), система (AI HR Assistant), LLM (внешняя система)\
**Предусловия:** вакансия успешно создана и сохранена в системе, загружено минимум 1 резюме кандидата, LLM доступна для обработки запросов

Сценарий:
1. Система анализирует резюме кандидатов по критериям вакансии
1. LLM выделяет ключевые навыки, опыт и определяет соответствие
1. Система рассчитывает скоринг по формуле: вес × совпадение
1. Система отображает ранжированный список кандидатов с рейтингами
1. Пользователь просматривает список и выбирает кандидатов для интервью


User Value:
- Объективная оценка без экспертизы 
- Фокус на лучших кандидатах

## UC4: Генерация персонализированных вопросов и плана интервью
**Акторы:** пользователь (директор/основатель), система (AI HR Assistant), LLM (внешняя система)\
**Предусловия:** пользователь выбрал минимум 1 кандидата из ранжированного списка, завершен анализ соответствия кандидатов, LLM доступна для обработки запросов

Сценарий:
1. Пользователь выбирает кандидатов и нажимает "Сгенерировать вопросы"
1. Система отправляет данные кандидатов и вакансии в LLM
2. LLM анализирует для каждого кандидата:
    - Пробелы в компетенциях
    - Возможные риски
    - Точки роста и развития
1. LLM генерирует список вопросов, адаптированных под конкретного кандидата и план беседы для каждого выбранного кандидата:
    - вопросы на проверку опыта;
    - вопросы на выявление пробелов;
    - кейсовые задания (небольшое практическое задание, основанное на реальных задачах вакансии);
    - вопросы на культурное соответствие (на основе анализа описания вакансии и компании);
1. Система отображает сгенерированные вопросы для каждого кандидата
1. Пользователь просматривает и утверждает вопросы

User Value:
- Профессиональная подготовка к интервью за минуты 
- Глубокая персонализация вопросов под конкретного кандидата

### UC5: Экспорт результатов
**Акторы:** пользователь (директор/основатель), система (AI HR Assistant)
**Предусловия:** успешно сгенерированы вопросы для выбранных кандидатов, пользователь просмотрел и утвердил вопросы, система готова к формированию документов

Сценарий:
1. Пользователь нажимает "Экспортировать результаты"
1. Пользователь выбирает формат экспорта (PDF по умолчанию)
1. Система формирует структурированный документ, содержащий:
    - Данные кандидата
    - Персонализированные вопросы
    - План беседы
    - Ключевые моменты для проверки
1. Система генерирует файл в выбранном формате
1. Система предоставляет файл для скачивания
1. Пользователь сохраняет бриф на устройство

User Value:
- Готовый профессиональный бриф для интервью в один клик
- Возможность поделиться результатами с командой или сохранить для архива
- Снижение риска упустить важные моменты во время собеседования

## Value Propositions:

"Первый инструмент найма, который понимает, что вы не рекрутер." \
Наш сервис за минуты находит лучших кандидатов, ранжирует резюме и готовит персональные вопросы для интервью.AI HR Assistant объединяет простоту агрегаторов, глубину анализа сервисов проверки и безопасность корпоративных систем в одном решении для основателей-не-рекрутеров. 

## North Star Metric:

75% успешно закрытых вакансий с помощью платформы (по отзывам пользователей через неделю).
