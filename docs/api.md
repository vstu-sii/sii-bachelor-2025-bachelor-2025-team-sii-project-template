# Product Reviews Analysis API

API для анализа отзывов о товарах с маркетплейсов

## Назначение API
**Product Reviews Analysis API** — это сервис для анализа отзывов о товарах с маркетплейсов. Он помогает пользователям получать аналитику по отзывам на их продукты.

## Основные возможности

### 1. Аутентификация
- **Регистрация/вход через Telegram** (`POST /auth/telegram`)
  - Использует Telegram Widget для входа
  - Возвращает токен доступа
  - Ограничение: 5 попыток за 15 минут

- **Выход из системы** (`POST /auth/logout`)
  - Завершает сессию пользователя
  - Требует Bearer-токен

### 2. Управление товарами
- **Получение списка товаров** (`GET /products`)
  - Возвращает проанализированные товары пользователя
  - Поддерживает пагинацию (до 50 товаров на страницу)
  - Требует авторизации

## Как работает

1. Пользователь входит через Telegram, получает JWT-токен
2. Токен используется для доступа к защищенным эндпоинтам
3. Пользователь может просматривать свои проанализированные товары
4. API поддерживает пагинацию для работы с большими списками

## Технические особенности

- Два сервера: продакшен и тестовый
- RESTful архитектура
- Bearer-аутентификация
- Rate-limiting для защиты от злоупотреблений
- Стандартизированные коды ответов

## Пример использования

```bash
# Аутентификация через Telegram
curl -X POST https://api.reviews-analysis.com/v1/auth/telegram \
  -H "Content-Type: application/json" \
  -d '{
    "id": 123456789,
    "first_name": "John",
    "username": "johndoe",
    "auth_date": 1600000000,
    "hash": "abc123..."
  }'

# Получение списка товаров
curl -X GET https://api.reviews-analysis.com/v1/products?page=1&limit=10 \
  -H "Authorization: Bearer YOUR_TOKEN"
