# Детальные UX-сценарии пользовательских потоков

## 1. Регистрация и вход через Telegram

### Happy Path
**Пользовательский контекст:** Новый пользователь хочет быстро начать работу с системой

| Шаг | Действие | Системный ответ | Микроинтеракции |
|-----|----------|-----------------|------------------|
| 1 | Открывает главную страницу | Показывает hero-секцию с CTA "Начать анализ" | Плавная анимация появления элементов |
| 2 | Кликает "Регистрация через Telegram" | Открывает Telegram Widget в модальном окне | Затемнение фона, плавное появление виджета |
| 3 | Подтверждает авторизацию в Telegram | Показывает индикатор загрузки "Создаем ваш профиль..." | Анимация спиннера с логотипом системы |
| 4 | - | Перенаправляет в личный кабинет | Плавный переход с fade-эффектом |

### Edge Cases
- **Автоматический вход:** При активной сессии - мгновенный redirect в ЛК
- **Повторная регистрация:** Toast-уведомление "Аккаунт уже существует" + redirect на вход

### Обработка ошибок
- **Telegram Widget недоступен:** 
  - Показ fallback сообщения
  - Кнопка "Повторить" с таймером 5 сек
  - Альтернативные варианты связи с поддержкой

## 2. Первый анализ товара

### Happy Path
**Пользовательский контекст:** Пользователь хочет проанализировать свой первый товар

| Шаг | Действие | Системный ответ | LLM взаимодействия |
|-----|----------|-----------------|-------------------|
| 1 | Вставляет ссылку WB | Валидирует URL в реальном времени | - |
| 2 | Нажимает "Начать анализ" | Показывает прогресс-бар с этапами | - |
| 3 | - | Парсинг отзывов (0-30%) | - |
| 4 | - | ИИ-анализ тональности (30-70%) | **LLM: Анализ sentiment + аспектов** |
| 5 | - | Генерация отчета (70-100%) | **LLM: Сводка и рекомендации** |
| 6 | - | Открывает детальную страницу товара | Показывает анимированные графики |

### Микроинтеракции
- **Валидация ссылки:** Зеленая галочка при корректном URL
- **Прогресс-анализ:** Подробные статусы каждого этапа
- **Готовность отчета:** Push-уведомление при долгой обработке

## 3. Сравнение анализов товара

### Happy Path
**Пользовательский контекст:** Владелец товара хочет отследить динамику показателей

| Шаг | Действие | Системный ответ | Визуальные элементы |
|-----|----------|-----------------|-------------------|
| 1 | Открывает страницу товара | Показывает историю анализов чекбоксами | Hover-эффекты на карточках анализов |
| 2 | Выбирает 2-3 анализа | Активрует кнопку "Сравнить" | Плавное появление кнопки |
| 3 | Нажимает "Сравнить" | Открывает comparative dashboard | Анимация раскрытия панели |
| 4 | - | Показывает сравнительные графики | **LLM: Автоматические инсайты** |
| 5 | Изучает отчет | Подсвечивает ключевые изменения | Tooltips с детализацией метрик |

### LLM Touchpoints
- **Генерация инсайтов:** "Заметили улучшение в качестве доставки на 15%"
- **Анализ трендов:** "Отрицательные отзывы сместились с качества на упаковку"
- **Рекомендации:** "Рекомендуем обратить внимание на частые жалобы по браку"

## 4. Запуск нового анализа по датам

### Happy Path
**Пользовательский контекст:** Регулярный мониторинг новых отзывов

| Шаг | Действие | Системный ответ | Оптимизации |
|-----|----------|-----------------|------------|
| 1 | Нажимает "Новый анализ" | Открывает date picker | Предлагает smart-периоды |
| 2 | Выбирает период | Валидирует доступность данных | Кэширование предыдущих запросов |
| 3 | Подтверждает запуск | Инкрементальный парсинг | **Только новые отзывы** |
| 4 | - | Обновляет статистику | **LLM: Дельта-анализ изменений** |

### Edge Cases
- **Нет новых отзывов:** Информационный баннер без запуска LLM
- **Частые запросы:** Rate limiting с понятным таймером

## Системные микроинтеракции

### Уведомления и обратная связь
- **Success:** Green toast с иконкой ✓
- **Warning:** Yellow toast с иконкой ⚠
- **Error:** Red toast с иконкой ✗ и действием "Повторить"

### Состояния загрузки
- **Skeleton screens** для контента
- **Progress indicators** для длительных операций
- **Optimistic updates** где возможно

### Визуальные переходы
- **Fade-in/out** между страницами
- **Slide animations** для модальных окон
- **Chart animations** для графиков
