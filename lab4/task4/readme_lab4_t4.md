# Задание 4: Оптимизация UX

## 🎨 Анимации и переходы

### Используемые библиотеки
- **Framer Motion** - Основная библиотека анимаций
- **CSS Transitions** - Простые переходы
- **Spring Physics** - Физические анимации
- **Gesture Animations** - Анимации жестов

### Типы анимаций
1. **Входные анимации** - Появление элементов
2. **Выходные анимации** - Исчезновение элементов
3. **Анимации состояний** - Изменение состояний
4. **Микроинтеракции** - Мелкие анимации взаимодействия
5. **Фоновые анимации** - Фоновые эффекты

### Принципы анимаций
- **Duration**: 200-500ms для взаимодействий
- **Easing**: `ease-out` для входа, `ease-in` для выхода
- **Stagger**: Последовательные анимации с задержками
- **Spring**: Для физических взаимодействий

## ⚡ Оптимизации производительности

### Bundle Optimization
- **Code Splitting** - Разделение кода на чанки
- **Tree Shaking** - Удаление неиспользуемого кода
- **Dynamic Imports** - Динамическая загрузка компонентов
- **Image Optimization** - Оптимизация размеров и форматов

### Rendering Optimization
- **Memoization** - Кэширование вычислений
- **Virtualization** - Рендеринг только видимых элементов
- **Debouncing** - Группировка частых обновлений
- **Throttling** - Ограничение частоты обновлений

### Network Optimization
- **HTTP/2** - Мультиплексирование запросов
- **Brotli Compression** - Сжатие ресурсов
- **CDN** - Распределение контента
- **Service Workers** - Кэширование и офлайн режим

import { 
  AnimatedButton, 
  LikeButton, 
  InteractiveCard,
  ConfettiEffect 
} from '@/ux/microinteractions'

function InteractiveUI() {
  const [showConfetti, setShowConfetti] = useState(false)
  
  return (
    <>
      <ConfettiEffect trigger={showConfetti} />
      
      <InteractiveCard
        hoverEffect={true}
        glowEffect={true}
        onClick={() => setShowConfetti(true)}
      >
        <h3>Интерактивная карточка</h3>
        <LikeButton count={42} />
        <AnimatedButton
          variant="primary"
          icon={<Sparkles />}
          onClick={() => setShowConfetti(true)}
        >
          Праздновать!
        </AnimatedButton>
      </InteractiveCard>
    </>
  )
}
