# Требования к моделям для оценки уборки помещений

## Выбор типов моделей
- **Основная LLM**: GPT-4V или LLaVA (мультимодальные модели)
- **Альтернативы**: BLIP-2, OpenFlamingo
- **Embeddings**: CLIP для семантического поиска похожих изображений

## Требования к производительности
- **Latency**: < 5 секунд на изображение
- **Throughput**: 20+ изображений в минуту
- **Availability**: 99% uptime

## Метрики качества
- **Accuracy**: согласованность с экспертной оценкой
- **ROUGE-L**: для оценки текстовых описаний
- **Custom metrics**: 
  - Score Accuracy (1-10 шкала)
  - Category Match (чисто/грязно/средне)
  - Hallucination Rate

## Ограничения
- **Память**: < 8GB RAM для инференса
- **Стоимость**: < $0.02 на изображение
- **Приватность**: локальная обработка или E2E шифрование
