# Baseline Model Report

## Метрики качества
- **Overall Accuracy**: 78%
- **Score Deviation**: ±1.2 points
- **Category Match**: 85%
- **Inference Time**: 3.2s avg

## Анализ ошибок
1. **Переоценка**: Модель завышает оценки для слабо освещенных помещений
2. **Недооценка**: Не замечает мелкий мусор в углах
3. **Hallucinations**: 5% случаев - вымышленные детали

## План улучшений
1. Добавить препроцессинг освещения
2. Обогатить датасет угловыми снимками
3. Реализовать confidence scoring
